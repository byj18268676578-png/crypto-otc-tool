# 📊 使用说明（简化版）

## 🚀 最简单的方法

### 1️⃣ 运行工具
```bash
python3 generate_from_text.py
```

### 2️⃣ 输入日期（可选）
```
请输入日期 (默认：2026-04-22): 
```
直接按回车使用当前日期

### 3️⃣ 粘贴数据
```
BTC 1200 150 进场期第 5 天
ETH 1300 180 进场期第 6 天
SOL 100 20 退场期第 3 天
END
```

### 4️⃣ 查看结果
打开 `dashboard.html` 文件即可！

---

## 📝 支持的格式

**空格分隔（推荐）：**
```
BTC 1200 150 进场期第 5 天
```

**逗号分隔：**
```
BTC: 1200, 150, 进场期第 5 天
```

**竖线分隔：**
```
BTC | 1200 | 150 | 进场期第 5 天
```

**表格格式：**
```
BTC 1200 150 进场期第 5 天
ETH 1300 180 进场期第 6 天
```
