🚀 使用方法

  1. 解压工具包：tar -xzf crypto-otc-tool.tar.gz
  2. 编辑 sample_data.json 填入你的数据
  3. 运行：python3 generate_from_sample.py
  4. 打开生成的 dashboard.html 查看可视化

📂 文件说明

  • sample_data.json - 数据输入模板
  • generate_from_sample.py - 数据生成脚本
  • dashboard.html - 示例可视化页面
  • QUICK_START.md - 快速开始指南
